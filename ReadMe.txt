      --- Steps to run Website---
1. Open VS Code.
2. Install Live Server extension.
3. Go to File option (Top Left).
4. Click open folder and select Event Management Folder.
5. When the folder is open in VS Code, just Go to Event.html file.
6. In Event.html file, right click and select(Open with live server).
7. After that the website interface is open.
